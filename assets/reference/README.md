# Reference assets (carried forward, not yet shippable)

Pre-PRD materials that, per `docs/product/PRD.md` §11.4, carry forward as the
style/direction basis for production rather than being discarded. Not shippable
in their current raw form:

- `animation-gifs/` — early movement-animation reference for the hero character carried forward in `assets/sprites/heroes/placeholder/`.
- `audio-demo/` — the demo music track that is the music-direction basis (PRD §11.2). Run it through `tools/gbmusic/` to produce a Game Boy chiptune draft, then hand-tune in LSDJ toward a shippable stem — see `tools/gbmusic/README.md`.
